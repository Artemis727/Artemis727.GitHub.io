关于pct_change()函数：pct_change主要涉及以下参数，

  • periods=1，用来设置计算的周期。

  • fill_method=‘pad’，如何在计算百分比变化之前处理缺失值(NA)。

  • limit=None，设置停止填充条件，即当遇到填充的连续缺失值的数量n时，停止此处填充

  • freq=None，从时间序列 API 中使用的增量（例如 ‘M’ 或 BDay())
